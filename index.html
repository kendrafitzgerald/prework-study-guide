Prework Study Guide

HTML

  • Creates structure of a web page
  • Provides content and basic framework for all pages on the web
  • Includes text, image, header, and list elements (types of content on a web page)
  • HTML provides a sturdy frame to your web page
  • HTML elements typically composed of opening tag or closing tag wihtin angle brackets, contain what we want to include in the element
  • head element contains info (metadata) that helps browser to render the page correctly, not visible when user looks at webpage
  • body elements contains content that is shown to the user, all elements we want the user to see and interact with must be placed inside this
  • !DOCTYPE html declaration lets browser know they should expect HTML on the page
  • html opening tag which shows the start of the HTML doc; attribute of lang=”en” tells browser the language for webpage
  • HTML attributes defines additional info about an element
  • head element contains metadata
  • Metadata is data about the webpage that browser uses to effectively serve webpage to users
  • title element is a metadata tag that defines title for the webpage
  • style is the CSS styles associated; base defines base URL; link connects external resource to HTML doc; meta defines metadata like character set, description, keywords, author and viewport
  • Script is client side scripts in the HTML doc
  • Head element defines following elements Charset (character encoding) Http-equiv (sets an HTTP header for content) Viewport (sets default view for our webpage)
  • Child elements are elements nested inside of other elements
  • Parent element is element that child elements are inside of
  • Body element defines body of document, holds content visible to user, all content we want to display must be inside of this
  • body element is an example of a semantic HTML element
  • Semantic HTML elements, name of element describes its content and the role it plays on a webpage; make HTML easier to read by other developers and browser
  • Non semantic elements div creates division in the page, span creates an inline division
  • Opening tags and closing tags, have the angle brackets, closing tags begin with a forward slash after first angle bracket
  • Elements are entire element from opening tag to closing tag; tag is only what appears inside the angle brackets
  • Headings are represented by h1, h2 etc, and represent larger or more prominent titles and subtitles to organize a page
  • p element represents a paragraph or block of text
  • ul=unordered list, ol=ordered list, li= list items (bulleted or numbered lists), ul contains an unordered bulleted list
  • img contains information about images that are displayed on the webpage, can contain attributes like src which defines location of the image
  • Alt attribute contains a text string that describes the appearance and functionality of the image
  • Br element creates line break between two blocks of content
  • a stands for anchor, creates linked to the same webpage or other webpages
  • Href attribute points to the URL for the link
  • Footer element goes at end of page, usually includes copyright and things like that
  • section elements are semantic elements that define a section in a document that contains thematic content
  • id and class are used to identify and select elements

CSS

  • Used to define how webpage content is presented to users
  • Defines colors, aesthetics, visual layout, and organization
  • Three types of styles: Inline, internal, external
  • Inline CSS are used when a developer wants to style an element directly in the HTML file
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only; useful for quick and permanent changes but not as flexible as external and internal style sheets (must be edited individually)
  • Inline CSS- style attribute to change color of h1 heading
  • Internal style sheet can be embedded directly into the HTML file, contains CSS rules for the webpage, rules only apply to that one page, and you can configure CSS classes and IDs to identify elements and style multiple elements in the page code
  • A single change to the CSS rule will apply to all tagged page elements
  • External CSS style sheet is most common way to style a website
  • External CSS style sheets allows developers to keep all CSS rules in a separate file which makes design changes easier
  • When using external style sheet, the file is linked using a link element placed in the HTMLs head (header)
  • CSS Rule: selector defines the element or attributes to which the declarations apply
  • Declarations contain two important components: CSS property we want to apply and the value of the property
  • Example: the declaration color: blue; includes the CSS property color and the value of that property, blue
  • Margin property indicates how much space we want around the outside of an element, not visible on page, adds a buffer, margin and 0 means no extra space
  • Padding property adds space around the content inside an element, increasing padding makes the entire element bigger because it is located inside an element
  • CSS box model: margins outside the element, elements borders, padding, and content
  • Asterisk will apply a rule to all elements visible on page
  • DRY (do not repeat yourself) Code is a principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines that need to be written or maintained: example is header, footer selectors with same declaration
  • Text align is a position property, can center a title or heading on a webpage
  • CSS rules can be applied to images as well, img selector assigns property values to any image element in the markup
  • Display property is assigned with the block value, which assigns each image to start on a new line
  • Link is needed to connect stylesheet to HTML, place it after final meta element in head, and before title element
  • The Rel attribute in the link specifies the relationship between the current document and the linked document resource
  • The href attribute specifies the URL of the external resource, it is using a relative path which means the location is derived from the current location, and to locate the style sheet from the current location of index.html we need to look in assets folder
  • Class attributes allow you to share a CSS rule to any element of choice by assigning the rule to a class attribute with a class selector; CA can be assigned to element in HTML and we can freely choose which elements are assigned the CSS rule
  • Adding a period before the card selector designates card as a class selector, any HTML element assigned to class card will gain these properties
  • HTML has color codes you can assign in CSS either by using semantic references or a hex # or hexademical code that is on HTML color codes website
