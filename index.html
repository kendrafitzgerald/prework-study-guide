HTML

  • Creates structure of a web page
  • Provides content and basic framework for all pages on the web
  • Includes text, image, header, and list elements (types of content on a web page)
  • HTML provides a sturdy frame to your web page
  • HTML elements typically composed of opening tag or closing tag wihtin angle brackets, contain what we want to include in the element
  • head element contains info (metadata) that helps browser to render the page correctly, not visible when user looks at webpage
  • body elements contains content that is shown to the user, all elements we want the user to see and interact with must be placed inside this
  • !DOCTYPE html declaration lets browser know they should expect HTML on the page
  • html opening tag which shows the start of the HTML doc; attribute of lang=”en” tells browser the language for webpage
  • HTML attributes defines additional info about an element
  • head element contains metadata
  • Metadata is data about the webpage that browser uses to effectively serve webpage to users
  • title element is a metadata tag that defines title for the webpage
  • style is the CSS styles associated; base defines base URL; link connects external resource to HTML doc; meta defines metadata like character set, description, keywords, author and viewport
  • Script is client side scripts in the HTML doc
  • Head element defines following elements Charset (character encoding) Http-equiv (sets an HTTP header for content) Viewport (sets default view for our webpage)
  • Child elements are elements nested inside of other elements
  • Parent element is element that child elements are inside of
  • Body element defines body of document, holds content visible to user, all content we want to display must be inside of this
  • body element is an example of a semantic HTML element
  • Semantic HTML elements, name of element describes its content and the role it plays on a webpage; make HTML easier to read by other developers and browser
  • Non semantic elements div creates division in the page, span creates an inline division
  • Opening tags and closing tags, have the angle brackets, closing tags begin with a forward slash after first angle bracket
  • Elements are entire element from opening tag to closing tag; tag is only what appears inside the angle brackets
  • Headings are represented by h1, h2 etc, and represent larger or more prominent titles and subtitles to organize a page
  • p element represents a paragraph or block of text
  • ul=unordered list, ol=ordered list, li= list items (bulleted or numbered lists), ul contains an unordered bulleted list
  • img contains information about images that are displayed on the webpage, can contain attributes like src which defines location of the image
  • Alt attribute contains a text string that describes the appearance and functionality of the image
  • Br element creates line break between two blocks of content
  • a stands for anchor, creates linked to the same webpage or other webpages
  • Href attribute points to the URL for the link
  • Footer element goes at end of page, usually includes copyright and things like that
  • section elements are semantic elements that define a section in a document that contains thematic content
  • id and class are used to identify and select elements
